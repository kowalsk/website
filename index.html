Setting up Win 11 on the go

1) Download Win 11 from MS portal 2) Download Rufus portable 3) Run Rufus as admin and install Windows 11 on USB HDD / SSD 4) Install Github Desktop, Visual Studio Code, Go, Paint.net, Minecraft, WhatsApp, TotalCMD, FreeFileSync, OneNote for Windows 5) Set up Wallpapers, viola!
